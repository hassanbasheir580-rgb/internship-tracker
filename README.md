# Internship Tracker

A full-stack web application that helps students organize and manage their internship applications throughout the recruitment process.

---

## Overview

Internship Tracker is designed to simplify the internship application journey by providing a centralized dashboard to manage applications, monitor progress, schedule interviews, and keep track of important deadlines.

Instead of relying on spreadsheets or scattered notes, users can store all internship-related information in one place and monitor every application from submission to final decision.
